# Multi Step Bootstrap Form with animations

- Pug/HTML
- Bootstrap 4.1.3
- SCSS/CSS
- JS (ES6)/ Babel

## Changing animation type

- change `data-animation` attributes value (on `.multisteps-form__panel`) to your preferred animation type;
- available animation values:
  - scaleIn
  - scaleOut
  - slideHorz
  - slideVert
  - fadeIn
